# VSCodeで日記を書こう(テンプレート)

このプロジェクトは、Qiitaに[記事](https://qiita.com/TanakaMikihisa/items/164a5056ab6035282fd0)として投稿した*VSCodeで日記を書こう！そしてAIに読ませよう！*の日記テンプレートです。

## ディレクトリ

| 場所 | 内容 |
| --- | --- |
| `daily/` | 日記（`YYYY-MM-DD.md` 形式） |
| `template.md` | 新しい日記を書き始めるときの雛形 |

## 拡張機能について

日記をより書きやすくするために`Daily Calendar`のインストールをお勧めしています。
サイドバーにカレンダーを表示でき、日にちをクリックすることで日記を生成できます。

## 整形とコミット

`make`で`daily/`または`*/daily/`配下の変更したMarkdownのフォーマットを整理し、コミットまで行います。

変更対象のMarkdownは**ファイル単位**で処理され、それぞれ整形のあと**1ファイルにつき1コミット**が作成されます。複数ファイルを変えた場合は、コミットもその数だけ積み上がります。

コミットメッセージは次のルールです（`display_name`はルートならファイル名のみ`仕事/daily/…`のようにネストしている場合は`仕事/ファイル名.md`のように親フォルダ付きで表示されます）。

- まだ Git に載っていないファイル: `☘️Add <display_name>`
- 既に追跡されているファイル: `✨Rewrite <display_name>`
